# 🎨 Gemini Image Maker

Google Chromeの拡張機能で、選択したテキストからGemini APIを使用して画像を生成します。

## ✨ 機能

- 📝 Webページ上のテキストを選択して右クリックメニューから画像生成
- 🖼️ Google Gemini APIを使用した高品質な画像生成
- 🆕 生成した画像を新しいタブで表示
- 📋 画像のクリップボードへのコピー機能
- 💾 画像のダウンロード機能
- ⚙️ 簡単なAPIキー設定画面

## 📋 必要要件

- Google Chrome ブラウザ
- Google Gemini API キー（[Google AI Studio](https://aistudio.google.com/app/apikey)から取得）

## 🚀 インストール方法

### 1. アイコン画像の生成

拡張機能を使用する前に、アイコン画像を生成する必要があります。

**方法A: 自動生成スクリプトを使用**
1. `create-placeholder-icons.html` をブラウザで開く
2. 自動的に3つのアイコンファイル（icon16.png, icon48.png, icon128.png）がダウンロードされます
3. ダウンロードされたファイルを `icons` フォルダに移動

**方法B: 手動生成**
1. `icons/generate-icons.html` をブラウザで開く
2. 「アイコンを生成」ボタンをクリック
3. 各サイズの画像を右クリックして保存
4. icon16.png, icon48.png, icon128.png として `icons` フォルダに保存

### 2. Chrome拡張機能としてインストール

1. Google Chromeを開く
2. アドレスバーに `chrome://extensions/` と入力してEnter
3. 右上の「デベロッパーモード」をONにする
4. 「パッケージ化されていない拡張機能を読み込む」をクリック
5. このフォルダ（`image-maker`）を選択

## ⚙️ 初期設定

### APIキーの設定

1. [Google AI Studio](https://aistudio.google.com/app/apikey)にアクセス
2. Googleアカウントでログイン
3. 「Create API Key」をクリック
4. 生成されたAPIキーをコピー
5. 拡張機能アイコンをクリック → 「⚙️ 設定を開く」
6. APIキーを入力
7. （オプション）プロンプトのプレフィックスとアスペクト比をカスタマイズ
8. 「💾 保存」をクリック

## 📖 使い方

1. **テキストを選択**
   - 任意のWebページで画像生成したいテキストを選択

2. **右クリックメニューから生成**
   - 選択したテキストを右クリック
   - 「選択テキストから画像を生成」を選択

3. **画像が表示される**
   - 新しいタブが開き、生成された画像が表示されます
   - 生成には数秒かかる場合があります

4. **画像を保存・コピー**
   - 「画像をコピー」ボタン: クリップボードにコピー
   - 「ダウンロード」ボタン: PNG形式で保存
   - または画像を右クリックして「画像をコピー」

## 🎯 使用例

選択するテキスト例:
- "夕暮れの富士山と桜"
- "未来都市のスカイライン"
- "かわいい子猫が遊んでいる"
- "抽象的な幾何学模様、青と紫のグラデーション"

画像生成時のプロンプト:
```
次の説明を一番わかりやすくするシンプルな画像を生成してください: [選択したテキスト]
```

## 🔧 トラブルシューティング

### APIキーが無効と表示される
- 設定画面でAPIキーが正しく入力されているか確認
- Google AI Studioで新しいAPIキーを生成
- APIキーの使用制限を確認

### 画像が生成されない
- インターネット接続を確認
- 設定画面でAPIキーが設定されているか確認
- ブラウザのコンソール（F12）でエラーを確認
- Gemini APIの利用制限に達していないか確認

### 拡張機能が表示されない
- `chrome://extensions/` でデベロッパーモードがONか確認
- 拡張機能が有効になっているか確認
- アイコン画像（icon16.png, icon48.png, icon128.png）が `icons` フォルダに存在するか確認

## 🔐 プライバシーとセキュリティ

- APIキーはChromeの同期ストレージに安全に保存されます
- 選択したテキストのみがGemini APIに送信されます
- 生成された画像はローカルに保存され、外部に送信されません
- ⚠️ **GitHubに公開する場合**: `.gitignore`が設定されているため、APIキーは含まれません

## 📝 注意事項

- Gemini APIの使用には料金が発生する場合があります
- APIの使用制限（レート制限、クォータ）に注意してください
- 生成される画像の品質や内容はGemini APIに依存します
