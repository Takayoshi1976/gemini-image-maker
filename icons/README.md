# アイコン画像の生成方法

このフォルダには拡張機能のアイコン画像を生成するためのファイルが含まれています。

## 生成方法

1. `generate-icons.html` をブラウザで開く
2. 「アイコンを生成」ボタンをクリック
3. 各サイズの画像を右クリックして「名前を付けて画像を保存」
4. 以下のファイル名で保存:
   - `icon16.png`
   - `icon48.png`
   - `icon128.png`

## または

オンラインのSVG to PNGコンバーターを使用:
1. `icon.svg` を開く
2. https://convertio.co/ja/svg-png/ などのサイトにアップロード
3. 16x16, 48x48, 128x128 の3サイズに変換
4. このフォルダに保存

## 代替方法

Canva、Figma、Photoshopなどのデザインツールで:
- グラデーション背景（#667eea から #764ba2）
- 画像アイコン（絵画のフレーム、山、太陽など）
を含む128x128pxのアイコンを作成し、16x16と48x48にリサイズ
